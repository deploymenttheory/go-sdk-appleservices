# Mock Data for Microsoft Mac Apps Version Tracker API

This directory contains mock JSON responses used for testing the Microsoft Mac Apps Version Tracker API client.

## Files

- `validate_latest_apps.json` - Sample response for GET /api/latest endpoint
- `responders.go` - HTTP mock responders for testing

## Data Source

The mock data is based on real API responses from <https://appledevicepolicy.tools/api/latest>
